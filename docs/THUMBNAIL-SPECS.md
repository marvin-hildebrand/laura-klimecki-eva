# Instagram Reel Thumbnail Spezifikationen

## Formate

| Zweck | Größe |
|-------|-------|
| Reel Upload | 1080 × 1920 px (9:16) |
| Safe Zone | 1080 × 1350 px (4:5 mittig) |
| Feed/Grid Crop | Mittlere 60% sichtbar |

## Safe Zone Regel

```
┌────────────────────┐
│  UNSICHER (oben)   │ ← wird im Grid abgeschnitten
├────────────────────┤
│                    │
│   HEADLINE (kurz)  │  ← Safe Zone
│                    │
│   GROSSES GESICHT  │  ← Safe Zone
│                    │
├────────────────────┤
│  UNSICHER (unten)  │ ← wird im Grid abgeschnitten
└────────────────────┘
```

**Alles Wichtige in die MITTE (mittlere 60%)**:
- Text
- Gesichter
- Headlines
- Logos

## Text-Regeln

- **Maximum 3-7 Wörter**
- **Sehr große Schrift** (YouTube Thumbnail Größe)
- **Großbuchstaben**
- **Fette Sans-Serif**

### Beispiele für kurze Headlines

Statt: "WARUM GUT AUSSEHENDE UNTERNEHMER ALS ERFOLGREICHER WAHRGENOMMEN WERDEN"

Besser:
- "WARUM ATTRAKTIVE CEOs MEHR VERTRAUEN"
- "SCHÖNE MENSCHEN VERKAUFEN BESSER"
- "LOOKS = STATUS?"
- "WARUM ERFOLG ATTRAKTIV WIRKT"

## Gesichter

- **Sehr nah** (Nahaufnahme)
- **Starke Emotion**
- **Klarer Blick**
- **Wenig Hintergrund**

## Farben (Laura CI)

| Element | Farbe |
|---------|-------|
| Hintergrund | Dunkelblau #1e3a5f |
| Keyword/Akzent | Hellblau #5ba4d4 |
| Text | Weiß #FFFFFF |

## Ideale Thumbnail-Struktur

```
┌────────────────────┐
│                    │
│   KURZE HEADLINE   │  ← Oben
│                    │
│   ┌────────────┐   │
│   │            │   │
│   │  GROSSES   │   │  ← Mitte (60%)
│   │  GESICHT   │   │
│   │            │   │
│   └────────────┘   │
│                    │
│   kleines Branding │  ← Unten
│                    │
└────────────────────┘
```

## Performance-Tipps

### Was funktioniert
- ✅ Dunkle Hintergründe
- ✅ Blaue Kontraste
- ✅ Weiße Schrift
- ✅ Nahaufnahme
- ✅ Hoher Kontrast
- ✅ Clean/Minimal

### Was NICHT funktioniert
- ❌ Zu viele kleine Wörter
- ❌ Komplexe Layouts
- ❌ Viele Elemente
- ❌ Dünne Schrift

## Export

| Einstellung | Wert |
|-------------|------|
| Format | JPG oder PNG |
| Qualität | 80-100% |
| DPI | irrelevant für Instagram |
